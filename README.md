Car Rental System

A simple and efficient Car Rental System built to manage car bookings, customers, and rental operations digitally. This project helps users rent cars easily while allowing admins to manage vehicles and bookings smoothly.


Features
🔐 User Authentication (Login / Signup)
🚘 Browse Available Cars
📅 Book Cars for Specific Dates
💳 Rental Price Calculation
📋 Booking Management
🛠️ Admin Panel for Car Management
❌ Cancel Bookings
📊 Rental Records & History
💾 Database Integration
