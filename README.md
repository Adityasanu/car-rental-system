# 🚗 Car Rental System

A simple and efficient Car Rental System built to manage car bookings, customers, and rental operations digitally.

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 🚘 Browse Available Cars
- 📅 Book Cars for Specific Dates
- 💳 Rental Price Calculation
- 📋 Booking Management
- 🛠️ Admin Panel for Car Management
- ❌ Cancel Bookings
- 📊 Rental Records & History
- 💾 Database Integration

---

## 🚀 Getting Started

```bash

```

